# Global Tech Layoffs Analysis Dashboard

> Interactive Power BI dashboard analyzing 383,000+ tech layoffs from 2020 to 2023

## 📸 Dashboard Preview

![Dashboard Screenshot](dashboard_screenshort.png)

## 📌 Project Overview

This dashboard was built to analyze the wave of global tech layoffs between 2020–2023. It provides insights into which companies, countries, and industries were most affected.

## 🔍 Key Insights

- **United States** accounted for the majority of all layoffs globally
- **Consumer** and **Retail** industries had the highest total layoffs
- Layoffs peaked in **early 2023** following the post-COVID hiring correction
- **116 companies** laid off 100% of their workforce entirely
- **Amazon, Google, Meta** were the top 3 companies by total layoffs

## 🛠️ Tools Used

- **Power BI Desktop** — dashboard design and visuals
- **MySQL** — data storage and cleaning (layoffs_staging2)
- **DAX** — measures: SUM, CALCULATE, DISTINCTCOUNT, DIVIDE

## 📊 Dashboard Features

- KPI Cards: Total Layoffs, Total Companies, Avg Layoffs, Full Layoff Companies
- Bar Charts: Layoffs by Country, Layoffs by Industry (Top 8)
- Line Chart: Monthly layoff trend over 3 years
- Top 10 Companies table with industry and country
- Year slicer to filter all visuals by year
